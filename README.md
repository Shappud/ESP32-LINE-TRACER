# ESP32 PID Line Tracer Robot (IN DEVELOPMENT)

> A high-speed autonomous line-following robot powered by an ESP32 using PID control and custom-designed hardware.

---

##  About

This repository documents the complete development of my ESP32-based line tracing robot.

Instead of only uploading the final code, this repository serves as an engineering journal showing the entire design process—from hardware selection and PCB design to PID tuning, testing, debugging, and competition preparation.

The goal is to build a fast, reliable, and competition-ready robot capable of navigating complex tracks with sharp turns using efficient control algorithms.

---

##  Hardware (add pics)

- ESP32 Microcontroller
- High-speed PID Line Following
- 5 IR Reflectance Sensors
- TB6612FNG Motor Driver
- Dual 6v 1000rpm N20 DC Gear Motors
- 11.1v 1400mAh Li-Po battery
- Custom Chassis Design
- PCB Design
